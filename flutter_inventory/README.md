## 🚀 الحصول على APK (خطوات بسيطة)

### الخطوة الأولى: رفع الكود على GitHub
> هذا تم بالفعل من Replit

### الخطوة الثانية: الانتظار (~10 دقائق)
GitHub سيبني APK تلقائياً

### الخطوة الثالثة: تنزيل APK
1. افتح الـ repository على GitHub
2. اضغط على **Actions** (في القائمة العلوية)
3. اضغط على أحدث تشغيل (الأول في القائمة)
4. انزل للأسفل حتى ترى **Artifacts**
5. اضغط على **مخزوني-APK** لتنزيل الملف
6. افتح الملف على هاتفك وثبّت التطبيق

> ⚠️ ملاحظة: عند تثبيت APK من خارج Google Play، قد يطلب منك الهاتف السماح بـ "تثبيت من مصادر غير معروفة" — هذا طبيعي وآمن.

## 🛠️ للمطورين: بناء يدوي

```bash

flutter build apk --release
```

APK موجود في: `build/app/outputs/flutter-apk/app-release.apk`
